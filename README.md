# Cerealbox
Arduino-based network monitor, based on the original by Steve Ocepek @ https://github.com/SpiderLabs/cerealbox

## Requirements
### Client & Server
- python = 2.7.x

### Client Python Packages
- pycapy >= 0.10.8
- impacket >= 0.9.6
- pygeoip >= 0.2.5
- incf.countryutils >= 1.0

### Server Python Packages
- pyserial >= 2.6
